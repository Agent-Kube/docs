## What is AgentKube?

AgentKube is an AI-powered SRE (Site Reliability Engineering) Agent that automatically investigates Kubernetes issues and generates compliance reports. Companies in regulated industries use AgentKube to reduce incident response time by 70% while maintaining comprehensive audit trails.

## Key Features

### AI-Powered Investigations

<AccordionGroup>
  <Accordion icon="magnifying-glass" title="Investigation">
    - Natural language processing for issue descriptions
    - Automatic response to Kubernetes alerts
    - Systematic information gathering
    - Multiple hypothesis testing
    - Root cause determination without pre-existing response protocols
  </Accordion>
  <Accordion icon="book" title="Response Protocol Automation">
    - Upload and manage existing response protocols
    - Connect response protocols to specific alerts
    - Automated response protocol execution
